
# Blockchain Coffee Traceability

## Overview
Ứng dụng blockchain truy xuất nguồn gốc cà phê từ nông trại đến người dùng cuối, sử dụng smart contract để đảm bảo minh bạch và chống giả mạo.

## Tech Stack
- **Smart Contract**: CoffeeTrace.sol (Solidity, deployed on Sepolia testnet)
- **UI**: Gradio interface for easy interaction
- **Development**: Google Colab notebooks
- **Frontend**: Web3.js/Alchemy for blockchain connection

## Quick Demo
Chạy Colab notebook để trải nghiệm Gradio UI: tra cứu nguồn gốc bằng batch ID.

## Setup & Deploy
1. Clone repo: `git clone https://github.com/LuanPham-44266/Blockchain-Coffee-Traceability`
2. Mở Colab: Import `CoffeeTrace.sol`, deploy to Sepolia (faucet: sepoliafaucet.com)
3. Chạy Gradio: `!gradio app.py`
